What I've done

CodeMelon

CodeMelon is a functional programming language created by me in Python. I started the project in November 2020. It started off as a simple math parser, and has now grown to support datatypes, conditional statements, functions and much more! The repository for Codemelon can be found here.

ChessEngine

ChessEngine is my attempt at creating a complete Single/Multi player Chess game engine in Java. Singleplayer Mode will have an AI implemented using the MiniMax algorithm, and the skill level of the AI will be roughly equal to that of a skilled human player. This project is currently in development, and the repository is linked here.

TicTacToe-AI

This project was my first successful attempt at implementing the MiniMax algorithm in Python. It is a console Tic-Tac-Toe game which can be played between 2 players, against an AI or against random computer moves. You can also pit an AI against an AI or Random against Random! The repository for this project can be found here.
